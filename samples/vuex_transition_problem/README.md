# 4章 Vuexのtips

Vuexとtransitionコンポーネントを使った場合の問題となる動作のサンプルコードです。

## デモ

https://vue-tips-support-page.mya-ake.org/demo/vuex_transition_problem/

## commands

### yarn

#### インストール

```
$ yarn
```

#### 動作確認

```
$ yarn serve
```

### npm

#### インストール

```
$ npm install
```

#### 動作確認

```
$ npm run serve
```
