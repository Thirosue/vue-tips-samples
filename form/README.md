## commands

### yarn

#### インストール

```
$ yarn
```

#### アプリケーションの動作確認

```
$ yarn serve
```

#### テストの実行

```
$ yarn test
```

### npm

#### インストール

```
$ npm install
```

#### アプリケーションの動作確認

```
$ npm run serve
```

#### テストの実行

```
$ npm run test
```
